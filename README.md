# Kamtape's Open Source Project
Kamtape's open source project is designed to everyone to intend it, Everyone is free to download or use it! Formerly in 2023.

# How to download?
In linux systems and WSL you can do it easily. In Windows... You need to install some servers. like XAMPP Or WAMPServer.

- Use Git Or Download it as an archive
- Import it to the /xampp/htdocs directory
- Follow the setup instructions and... BOOM! you are done.
